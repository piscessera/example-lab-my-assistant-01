# Nvidia Stock Week Trend - Requirements

## Overview
A mini web application to check and display the weekly trend of NVIDIA (NVDA) stock.

## Core Features
1.  **Stock Data Fetching**: Retrieve NVIDIA stock data for the past consecutive week (7 days or 5 trading days).
2.  **Trend Visualization**: Display the data as a simple trend chart (line chart) or a clean list of daily changes.
3.  **Current Price**: Show the latest available price.
4.  **Responsive Design**: A clean, modern interface that works on desktop and mobile.

## Technical Stack
-   **Frontend**: HTML, CSS (Vanilla), JavaScript.
-   **Data Source**: Yahoo Finance API. *Note: Will likely require a CORS proxy for client-side access.*

## Non-Functional Requirements
-   Fast load time.
-   "Wow" factor design (dark mode, neon greens for Nvidia theme).
